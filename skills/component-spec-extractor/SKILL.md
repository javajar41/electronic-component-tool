# 电子物料规格提取工具

## 功能

1. **器件分类** - 支持电阻、电容、电感、IC、连接器、晶振等
2. **规格提取** - 自动识别型号、阻值/容值、精度、封装等
3. **描述生成** - 生成标准化的ERP料号描述
4. **批量处理** - 支持多条记录同时处理
5. **导出功能** - 支持CSV导出

## 使用方法

1. 选择物料分类（电阻/电容/电感/IC等）
2. 粘贴数据手册内容
3. 点击"提取规格"
4. 复制或导出结果

## 支持的格式

- 型号：RC0402、CL0603、STM32F103等
- 阻值：10KΩ、4.7K、1MΩ等
- 容值：100nF、10μF、100pF等
- 封装：0201、0402、0603、0805、1206等

## 技术实现

- 正则表达式匹配
- 本地存储配置
- 响应式UI设计

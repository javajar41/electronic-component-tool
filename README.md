# 电子物料规格描述工具

一个帮助电子工程师快速生成标准化电子元器件规格描述的Web工具。

## 🎯 功能特性

- **支持6种元器件类型**：电阻、电容、电感、二极管、三极管、IC芯片
- **智能表单**：根据元器件类型动态显示相关参数
- **标准描述生成**：一键生成符合行业标准的规格描述
- **历史记录**：本地保存生成的描述，方便查看和复用
- **响应式设计**：支持桌面和移动设备

## 🚀 使用方法

1. 打开 `index.html` 文件
2. 选择元器件类型
3. 填写相关参数
4. 点击"生成规格描述"
5. 复制生成的描述使用

## 📋 支持参数

### 电阻
- 阻值、精度、功率、封装、品牌、型号

### 电容
- 容值、耐压、精度、类型、封装、品牌、型号

### 电感
- 电感值、额定电流、精度、封装、品牌、型号

### 二极管
- 类型、反向电压、正向电流、封装、品牌、型号

### 三极管
- 类型(NPN/PNP/MOSFET)、耐压、电流、功率、封装、品牌、型号

### IC芯片
- 功能类型、工作电压、封装、引脚数、品牌、型号

## 🛠️ 技术栈

- HTML5
- CSS3 (现代布局、动画效果)
- JavaScript (ES6+)
- LocalStorage (本地存储)
